# API Key Setup Guide

## Overview

Neothera uses **OpenRouter API** to power the AI Coach feature, which generates personalized insights based on your acne patterns and habits. This guide shows you how to set up the API key.

---

## Step 1: Get Your OpenRouter API Key

1. **Visit OpenRouter**: Go to [https://openrouter.ai](https://openrouter.ai)
2. **Sign Up/Log In**: Create an account or log in if you already have one
3. **Navigate to Keys**: Click your account icon → **Keys** (or go directly to credentials section)
4. **Create API Key**: Click "Create Key" and give it a name like "Neothera"
5. **Copy the Key**: You'll see your API key displayed—copy it (it starts with `sk-or-`)

---

## Step 2: Add to Your Environment

### Option A: Local Development (.env.local)

1. **Create `.env.local` file** in your project root (`Neothera/` folder)
2. **Add this line**:
   ```
   OPENROUTER_API_KEY=your-api-key-here
   ```
   Replace `your-api-key-here` with the key you copied from OpenRouter

3. **Example** (do not use this key; it's fake):
   ```
   OPENROUTER_API_KEY=sk-or-1234567890abcdefghijklmnopqrstuvwxyz
   ```

4. **Restart your dev server** if it's running:
   ```bash
   npm run dev
   ```

### Option B: Vercel Deployment

1. **Go to Vercel Dashboard** → Your Neothera project
2. **Settings** → **Environment Variables**
3. **Add Variable**:
   - Key: `OPENROUTER_API_KEY`
   - Value: Your OpenRouter API key
4. **Deploy** your changes to activate

### Option C: Docker / Production

Set the environment variable before starting:
```bash
export OPENROUTER_API_KEY=your-api-key-here
npm run dev  # or docker run, etc.
```

---

## Step 3: Verify It's Working

1. **Start the app**: `npm run dev`
2. **Open the dashboard**: [http://localhost:3000](http://localhost:3000)
3. **Click the Coach tab** (4th tab in the tab bar)
4. **Look for AI insights**: If insights appear with real analysis, your API key is working!

---

## What Happens Without an API Key?

If `OPENROUTER_API_KEY` is **not set**:
- ✅ Dashboard still works
- ✅ Insights, Timeline, Patterns tabs work normally
- ⚠️ Coach tab shows **fallback suggestions** (generic templates, not data-driven)
- ℹ️ You'll see a message indicating fallback mode is active

## Troubleshooting

### API Key Not Working
- **Check key is correct**: Verify you copied the full key (including `sk-or-` prefix)
- **Check `.env.local` format**: Ensure no extra spaces or quotes
- **Restart dev server**: `npm run dev` must be restarted after adding `.env.local`
- **Check OpenRouter account**: Ensure your account has credits/plan active

### 401 Unauthorized
- Your API key is invalid or expired
- Generate a new key from OpenRouter dashboard
- Update `.env.local` with the new key

### 429 Rate Limited
- You've hit API rate limits
- Wait a few minutes before retrying
- Consider upgrading your OpenRouter plan for higher limits

---

## Security Best Practices

⚠️ **Never commit `.env.local` to Git!**
- Add `.env.local` to your `.gitignore` file:
  ```
  .env.local
  .env.*.local
  ```
- Keep your API key secret—treat it like a password
- If you accidentally commit it, regenerate your key immediately in OpenRouter dashboard

---

## What Data Is Sent to OpenRouter?

When you use the AI Coach, the following data is sent to OpenRouter's API:
- Your 30-day acne severity trend (numbers only, no identifiable info)
- Detected habit patterns (correlations, confidence levels)
- Your top recommendations (generated by our analysis engine)
- **No personal data** (name, location, email, etc.) is sent

All communication is encrypted (HTTPS) and follows OpenRouter's privacy policy.
