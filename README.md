# Neothera – Skin Intelligence Dashboard

## 🚀 Overview
Neothera is an AI-driven habit tracking and skin intelligence platform that analyzes lifestyle patterns (sleep, stress, skincare, etc.) to generate actionable insights for improving skin health.

## ✨ Features
- 📊 Habit tracking with timeline view
- 🧠 AI-powered insights & recommendations
- 📈 Trend and severity analysis
- 🔍 Pattern detection engine
- 🎯 Personalized skin improvement suggestions

## 🛠 Tech Stack
- React / Next.js
- Tailwind CSS
- TypeScript
- Deployed on Vercel

## 📂 Project Structure
- `components/` – reusable UI components
- `src/app/` – Next.js pages and API routes
- `src/features/` – domain logic
- `src/utils/` – helper functions
- `src/hooks/` – custom React hooks
- `src/types/` – TypeScript definitions

## ⚙️ Setup
```bash
git clone https://github.com/AEX-WEX/neothera-skin-intelligence.git
cd neothera-skin-intelligence
npm install
npm run dev
```

Visit `http://localhost:3000` to see the application.
